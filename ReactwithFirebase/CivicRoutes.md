Civic Routes (Advanced React + Firebase Project)
Project Title

Civic Routes – Smart Civic Issue Reporting Platform

Project Overview

Civic Routes is a modern web application that allows citizens to report civic issues such as potholes, garbage, broken streetlights, or water leakage directly to local authorities. The platform enables users to submit issues with location details, images, and descriptions. Other users can view reported issues and track their resolution status.

The project is built using React for the frontend and Firebase for authentication, database, and storage, making it a scalable and real-time civic engagement platform.

The goal of the application is to create a transparent communication bridge between citizens and municipal authorities, ensuring faster problem resolution and better community participation.
