TaskFlow – Simple Task Manager

Project Overview

TaskFlow is a beginner-friendly productivity web application that helps users organize their daily tasks. Users can create, update, delete, and manage tasks efficiently. The application stores tasks in Firebase Firestore, allowing data persistence and real-time updates.

This project is ideal for beginners learning React state management and Firebase integration.

Key Features
🔐 User Authentication
Login and signup using Firebase Authentication
Each user manages their own tasks
✅ Task Management

Users can:

Add new tasks
Edit tasks
Delete tasks
Mark tasks as completed
📂 Task Categories

Tasks can be organized into categories like:

Work
Personal
Study
📊 Task Status

Each task can have:

Pending
Completed
🔄 Real-Time Updates
Tasks update instantly using Firebase Firestore
