# **Real Estate and Accommodation System**

## Context :

You and your friends have recently joined a dynamic startup that aims to transform the real estate and accommodation market in Bangalore. The bustling city is known for its rapid growth and high demand for housing, but finding suitable accommodation can be a nightmare due to limited information, outdated listings, and unreliable contacts.

## Project Goal :

- The UI/UX should be best in class
- Develop a Real Estate and Accommodation Application that simplifies the process of finding and renting properties in Bangalore.
- This application should allow users to search for houses and PG accommodations, view detailed listings with photos and reviews, and contact landlords directly.
- It should also include features to help landlords manage their properties and communicate with potential tenants.

## Minimum Expected Features :

- User Authentication : use services like Firebase or supabase for user authentication.
- **Property Listings**:
    - Allow landlords to create and manage property listings with photos, descriptions, amenities, and pricing.
    - Implement a robust search using binary search and filter system and sorting systems using algorithms likes quicksort, for users to find properties based on location, budget, and other preferences.
- **Searching and suggestions:** Search recommendations using edit distance algorithm would be plus point with debouncing
- **User Reviews and Ratings**:
    - Enable tenants to leave reviews and ratings for properties and landlords.
    - Display average ratings and recent reviews on property listings.
- **Contact and Communication**:
    - Provide in-app messaging or contact forms to facilitate communication between tenants and landlords.
    - Include options for scheduling property visits or virtual tours.
- **Admin Side**:
    - **User Management**: Handle user profiles, verification, and authentication.
    - **Property Analytics Dashboard**: Provide landlords with insights on listing views, inquiries, and tenant feedback.
    - **Support and Dispute Resolution**: Offer a platform for handling user support and resolving disputes.
- **Payment feature for users** : Integrate Razorpay API , to implement payment feature.
- **Database solution :**  You need to use database solution like Firebase , Supabase for this whole application.
- **Market-Beating Feature**: Implement an AI-based recommendation engine that suggests properties to users based on their search history, preferences, and past interactions, enhancing the likelihood of finding a suitable match quickly.
- **Bonus:** Finding and filtering closest properties using shortest path algorithm, you use react custom hooks to implement these logics

## Submission Guideline:

- A functional web application with source code.
- Comprehensive documentation including architecture, API details, and user guide.
- A presentation demonstrating the application and highlighting how it improves the housing search experience over existing solutions.
